# Register

| Register | ABI | Convention                  |
| -------- | --- | --------------------------- |
| x0       | go  | scope reference             |
| x1       | f1  | argument 1 or argument list |
| x2       | f2  | argument 2                  |
| x3       | f3  | argument 3                  |
| x4       | f4  | argument 4                  |
